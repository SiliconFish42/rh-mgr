# Introduction

**ROM Hack Manager (rh-mgr)** is a modern desktop application designed to simplify the experience of playing Super Mario World ROM hacks.

Instead of manually searching for hacks, downloading patches, finding your clean ROM, using a patching tool, and then organizing your files, **rh-mgr** handles it all.

## Key Features

- **Library Management**: Keep all your hacks in one place.
- **Integrated Discovery**: Browse SMW Central's database without leaving the app.
- **One-Click Patching**: The app handles IPS/BPS/UPS patching automatically.
- **Seamless Launching**: Integration with your favorite SNES emulator.

## Why use rh-mgr?

If you enjoy playing SMW hacks, you know the process can be tedious. **rh-mgr** aims to make it as easy as using a modern game launcher like Steam.
