# Library Management

The **Library** tab is your collection of patched and ready-to-play hacks.

## Viewing Hacks

- **Grid View**: Shows cover art (if available).
- **List View**: Shows detailed information in a table.

## Filtering and Sorting

Use the toolbar to organize your collection:

- **Search**: Filter by name.
- **Sort**: Order by Name, Date Added, Rating, etc.

## Launching a Game

1. Hover over a game card (or select it in list view).
2. Click the **Play** button.
3. The game will launch in your configured emulator.

## Deleting Hacks

To remove a hack from your library (and delete the patched file):

1. Right-click the game card.
2. Select **Delete**.
3. Confirm deletion.
