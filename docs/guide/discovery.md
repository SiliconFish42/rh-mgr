# Discovery & Patching

The **Discovery** tab connects directly to SMW Central's database to help you find new hacks.

## Finding Hacks

- Browse the latest releases.
- Use the search bar to find specific titles.
- Filter by difficulty (Easy, Normal, Kaizo, etc.).

## Patching a Hack

When you find a hack you want to play:

1. Click the **Download & Patch** button on the hack's card.
2. The application will:
   - Download the patch file.
   - Apply it to your clean base ROM.
   - Save the result to your library.
   - Fetch metadata and images.

Once finished, the button will change to **In Library**, and you can find it in your Library tab.

## Supported Formats

The internal patcher supports:
- **IPS**
- **BPS**
- **UPS**
